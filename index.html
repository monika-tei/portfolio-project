Design for Developers

An overview of the KEA's 3rd semester Front-End Design elective theme

Key notes on the course

Design for Developers was the second theme of the Front-End elective, and was more focused on how to write code for designing web interfaces. We learned some useful methods in how to create styling systems, what to know about the UI components, making things interact and finally - build responsive applications.


Over a time period of 1 week, we built 3 small projects related to a specific topic within web design, that can be found below:

Mini-Assignments

Fonts and UI components

For this project we took a simple landing page design and had to create a font-hierarchy and design the UI components. We got familiar with using REM values for type based on a Type Scale tool, as well as importing fonts from different sources (such as Google fonts and fonts from other sources).

Working with colors and animations

Learned some useful tools for finding & implementing color gradients, evaluating contrast levels, building color palette, placing global colors in the :root selector. In addition to that, we experimented with writing code for dark theme and how the colours need to switch.

Responsive design

Focusing mostly about using Grid and CSS flex, as well as min width/height, @media queries to layout the page. For this small project, it was about pairing text and images and finding a way to place them together for both desktop and mobile viewing.
